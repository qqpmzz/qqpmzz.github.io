# 📝 내 개발 블로그

이 저장소는 Jekyll을 사용한 GitHub Pages 블로그입니다.

## 🚀 특징

- **Markdown 기반** 글쓰기
- **자동 배포** (GitHub Pages)
- **반응형 디자인**
- **SEO 최적화**
- **코드 하이라이팅**

## 📁 구조

```
├── _posts/          # 블로그 글 (Markdown)
├── _config.yml      # Jekyll 설정
├── about.md         # 소개 페이지
├── archive.md       # 글 목록 페이지
└── assets/css/      # 커스텀 스타일
```

## ✍️ 글 작성법

`_posts/` 폴더에 `YYYY-MM-DD-제목.md` 형식으로 파일을 만들고:

```markdown
---
layout: post
title: "글 제목"
date: 2025-09-26 15:00:00 +0900
categories: [카테고리]
tags: [태그1, 태그2]
---

여기에 마크다운으로 내용을 작성하세요!
```

## 🎨 커스터마이징

- `_config.yml`: 사이트 기본 설정
- `assets/css/style.scss`: 디자인 커스터마이징
- 테마 변경: `_config.yml`의 `theme` 값 수정
